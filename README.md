## Aula 97
    Operador Spread em javascript
*vetor sem spread operator*

    const notas = [60, 45, 78, 80, 32, 86]

    console.log(notas);
    
*saida* 
    [ 60, 45, 78, 80, 32, 86 ]
    
*vetor com spread operator*
    
    const notas = [60, 45, 78, 80, 32, 86]

    console.log(notas);
*saída* 
    60 45 78 80 32 86
